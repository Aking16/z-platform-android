## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

1. Run the Z-Platform back-end by cloning [this repository](https://github.com/Aking16/z-platform). **Use the `android-backend` branch**

*This Project was made with postgres as the database, [configure Prisma](https://www.prisma.io/docs/getting-started/setup-prisma/add-to-existing-project) if you plan to use a different database*

2. Sync the project in the Android Studio!
